# Example phyex set old

Arabidopsis thaliana embryogenesis dataset from Xiang et al. 2011. check
the phyexSet package for more details

## Usage

``` r
example_phyex_set_old
```

## Format

A BulkPhyloExpressionSet with 7 developmental stages (1 rep each) and
25096 genes

## Source

phyexSets::Athaliana.embryogenesis_2011 matched with the Arabidopsis
thaliana phylomap from phylomapr
