# Calculate Transcriptomic Divergence Index (TDI)

Calculate the transcriptomic divergence index values for a
PhyloExpressionSet. This function provides backward compatibility with
the old TDI() function.

## Usage

``` r
TDI(phyex_set)
```

## Arguments

- phyex_set:

  A PhyloExpressionSet object

## Value

Numeric vector of TDI values for each identity
