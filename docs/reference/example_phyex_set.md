# Example phyex set

Arabidopsis thaliana embryogenesis dataset from Hoffman et al. 2019.
check the phyexSet package for more details

## Usage

``` r
example_phyex_set
```

## Format

A BulkPhyloExpressionSet with 8 developmental stages (3 reps each) and
27520 genes

## Source

phyexSets::Athaliana.embryogenesis_2019 matched with the Arabidopsis
thaliana phylomap from phylomapr
