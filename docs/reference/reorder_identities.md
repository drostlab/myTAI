# Set Identity Order for ScPhyloExpressionSet

Change the order of the identities (factor levels) in the Seurat object.

## Usage

``` r
reorder_identities(phyex_set, new_order)
```

## Arguments

- phyex_set:

  A ScPhyloExpressionSet object

- new_order:

  Character vector specifying the new order of identities

## Value

ScPhyloExpressionSet object with updated identity order
