# Calculate Transcriptomic Evolutionary Index (TEI)

Calculate the transcriptomic evolutionary index values for a
PhyloExpressionSet. This function provides backward compatibility with
the old TEI() function.

## Usage

``` r
TEI(phyex_set)
```

## Arguments

- phyex_set:

  A PhyloExpressionSet object

## Value

Numeric vector of TEI values for each identity
