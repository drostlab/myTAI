# Set Identities for ScPhyloExpressionSet

Change the Seurat identities to a different metadata column.

## Usage

``` r
set_identities(phyex_set, identity_name)
```

## Arguments

- phyex_set:

  A ScPhyloExpressionSet object

- identity_name:

  Character, name of the metadata column to use as new identities

## Value

ScPhyloExpressionSet object with updated identities
