# Calculate Transcriptomic Polymorphism Index (TPI)

Calculate the transcriptomic polymorphism index values for a
PhyloExpressionSet. This function provides backward compatibility with
the old TPI() function.

## Usage

``` r
TPI(phyex_set)
```

## Arguments

- phyex_set:

  A PhyloExpressionSet object

## Value

Numeric vector of TPI values for each identity
