# Transcriptomic Index Name Mapping

Named list mapping transcriptomic index abbreviations to full names.

## Usage

``` r
TI_map
```

## Format

A named list with 5 elements:

- TXI:

  Transcriptomic Index

- TAI:

  Transcriptomic Age Index

- TDI:

  Transcriptomic Divergence Index

- TPI:

  Transcriptomic Polymorphism Index

- TEI:

  Transcriptomic Evolutionary Index
